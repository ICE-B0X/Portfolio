Lingo Match

A coworker asked if I could develop a user-friendly solution for an educational matching game, specifically designed for language learning.

The concept is to create an application that allows users to upload a .csv file containing word pairs, which the app would then shuffle and present in a dynamic, interactive format. This way, users could continuously practice and match words, helping them to reinforce vocabulary in an engaging way.
